# CollegeRestro - Restaurant Landing Page

## Overview
CollegeRestro is a modern restaurant landing page built using HTML, CSS, and JavaScript.

## Features
- Responsive navigation bar
- Hero section with attractive UI
- Interactive food image gallery
- Food selection using JavaScript DOM manipulation
- Modern restaurant-themed design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure

```
restaurant-website/
│
├── index.html
├── styles.css
├── README.md
└── images/
```

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
